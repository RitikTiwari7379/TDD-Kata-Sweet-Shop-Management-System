# Screenshots

This directory will contain screenshots of the application once it's running.

## Required Screenshots

1. **login.png** - Login page showing authentication form
2. **register.png** - Registration page
3. **dashboard.png** - Main dashboard with sweet listings
4. **search.png** - Search functionality demonstration
5. **add-sweet.png** - Admin modal for adding sweets
6. **purchase.png** - Purchase confirmation

## How to Generate Screenshots

1. Start both backend and frontend servers
2. Navigate through the application
3. Take screenshots using your preferred tool
4. Save them in this directory with the above names
5. Update README if needed
